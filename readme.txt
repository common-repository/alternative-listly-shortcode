Makes embeding content from List.ly easier
== Description ==

<strong>Alternative List.ly shortcode plugin contains</strong> -
<ul>
<li>Easier embed of List.ly content.</li>
<li>Button "List.ly" within TinyMCE editor that outputs [listly_shortcode url="" number_per_page_listly="12"]</li>
<li>The admin user can embed lists from List.ly simply by pasting the URL address of the list between ""</li>
<li>The admin user can easily set the number of items per page, by changing the default value of 12(or change the default value by editing shortcode.js file)</li>
<li>The plugin doesn't give option for choosing the type of display for the list(Gallery, Minimal, Magazine, List), neither for showing the header or the author. These options are hardcored in the plugin. The type of display is set to <strong>Gallery</strong> and the other two options are disabled.</li>
<li>A simple and documented filter that provide the functionality.</li>
</ul>


== Installation ==

1. Upload `alternative-list-ly.zip` onto your local computer.
2. Go to your WordPress Dashboard and select <strong>Plugins >> Add New</strong>.
3. Click on the <strong>Upload</strong> option at the top and select the `alternative-list-ly.zip file` you just downloaded.
4. Click on <strong>Install</strong>.
5. Activate the plugin through the 'Plugins' menu in WordPress
6. Within your TinyMCE editor click on List.ly text button and paste URL of List.ly list between ""
